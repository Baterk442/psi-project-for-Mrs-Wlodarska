# Monika Sieradzka Beauty

Prosta strona internetowa salonu kosmetycznego wykonana w HTML, CSS, JavaScript i Bootstrap 5.

## Opis

Projekt przedstawia stronę typu landing page dla salonu kosmetycznego.  
Zawiera sekcję powitalną, opis firmy, galerię w formie karuzeli, cennik usług oraz formularz kontaktowy.

## Funkcje

- Responsywne menu nawigacyjne
- Sekcja hero ze zdjęciem
- Sekcja „O firmie”
- Galeria jako karuzela Bootstrap
- Cennik usług
- Formularz kontaktowy z prostą walidacją
- Automatyczne ustawianie usługi po kliknięciu przycisku „Wybieram”

## Technologie

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
